# 🎬 Netflix Clone — "Quem está assistindo?"

Um projetinho que desenvolvi para treinar e evoluir minhas habilidades em **Front-end**, recriando a famosa tela de perfis da Netflix.

A ideia foi focar não só na aparência, mas também na **experiência do usuário**, deixando tudo fluido, organizado e agradável de usar.

---

## 🚀 Tecnologias

Usei o básico bem feito (e com atenção aos detalhes):

- **HTML5** → estrutura da aplicação  
- **CSS3** → layout, responsividade e animações  
- **JavaScript (Vanilla)** → interatividade e manipulação do DOM  

---

## ✨ Funcionalidades

- 👤 Seleção de perfis estilo Netflix  
- 🎯 Efeitos de *hover* suaves (escala + destaque)  
- ➕ Adição dinâmica de novos perfis  
- ✏️ Interações simples de edição  
- 📱 Layout adaptável para diferentes telas  

---

## 🧠 Aprendizados

Esse projeto me ajudou a reforçar:

- Organização entre **HTML, CSS e JS**  
- Uso de **Flexbox** para layout e centralização  
- Manipulação do **DOM** com JavaScript  
- Melhoria na **experiência do usuário (UX)**  
- Escrita de código mais limpo e organizado
